# 🕒 Clock Store Pro (E-commerce Website)

## 📌 Project Description
Clock Store Pro is a simple and responsive e-commerce website built using **HTML, Tailwind CSS, and JavaScript**.  
This project allows users to **buy and sell clocks online** with features like cart management, live clock display, and local storage.

---

## 🚀 Features

- ⏰ **Live Digital Clock**
  - Displays real-time current time on the homepage.

- 🛒 **Shopping Cart System**
  - Add products to cart
  - Remove items from cart
  - Cart count updates dynamically

- 💾 **LocalStorage Integration**
  - Cart and product data are stored in browser storage
  - Data remains even after page refresh

- 📦 **Sell Product Feature**
  - Users can add their own clock products
  - Product name, price, and image URL supported

- 🎨 **Modern UI with Tailwind CSS**
  - Clean and responsive design
  - Hover animations and smooth effects

- 📱 **Responsive Design**
  - Works on mobile, tablet, and desktop devices

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS (CDN)
- JavaScript (Vanilla JS)
- LocalStorage API
